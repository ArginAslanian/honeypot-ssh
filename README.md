# Cowrie SSH Honeypot on Ubuntu

This project documents my setup of a **Cowrie SSH honeypot** on **Ubuntu Server** to simulate a fake SSH service, capture login attempts, and monitor attacker behavior in a controlled lab environment.

## What this project does
- Deploys **Cowrie** as an SSH honeypot
- Runs under a dedicated non-root user
- Uses a Python virtual environment for installation
- Logs attacker activity for analysis

## Purpose
The goal of this project is to practice honeypot deployment, and security monitoring by safely attracting and observing unauthorized SSH connection attempts.

## Tech Used
- Ubuntu Server
- Cowrie
- Python 3
- Git
- Bash
